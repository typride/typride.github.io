Hex

Data Analysis & BI active ★ 4.5 freemium · $75/month Free tier available

Hex is a collaborative data workspace for building and sharing data projects. It combines a reactive notebook environment with native app-building and AI agents in a single platform, enabling data teams to work with or without code to explore data, build models, and share insights as interactive web apps.
